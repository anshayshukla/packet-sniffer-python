# Packet Sniffer using Python

## Overview
This project captures and analyzes network packets using Python.

## Features
- Captures live network traffic
- Displays packet summary
- Helps understand TCP/IP protocols

## Tech Stack
- Python
- Scapy

## How to Run
1. Install scapy:
   pip install scapy
2. Run sniffer.py

## Note
This project is created for learning and demonstration purposes.
